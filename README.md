# Soldat scripts
I created this repository so I can upload Soldat scripts in a single repository.  
The idea is simple, if the script is just a small project, I will upload it here, if it's a more serious one, it will have a separate repository.


### Current Scripts
Open the script folder to see the respective description in the README file.
 - **SaveAndTele**
	 This script is intended to help on mapmaking and debuging. It allows infinite saves(making a new save replaces the previous one), and the teleport feature is similar to GodMode, you can teleport to the mouse cursor position
